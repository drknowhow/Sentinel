# Chrome Web Store Listing — Sentinel

## Short Description (132 chars max)
Record web interactions to auto-generate step-by-step HTML guides, run visual playback tests, and track bugs — all locally in your browser.

## Detailed Description

Sentinel is a lightweight browser extension for QA teams, developers, and anyone who needs to document or test web applications — without leaving the browser.

**Record & Replay**
Click "Record" and use your web app normally. Sentinel captures every click, keystroke, scroll, and navigation — generating smart CSS selectors and human-readable descriptions for each step. Play sessions back at adjustable speed or step through them one at a time.

**Auto-Generate Visual Guides**
Turn any recorded session into a polished, standalone HTML guide complete with embedded screenshots. Use the built-in editor to reorder steps, add titles and notes, toggle screenshots, and write an introduction — then export and share with your team.

**Test with Assertions**
Add assertions to any step — check that elements are visible, hidden, contain specific text, or have a CSS class. Run playback and get a clear pass/fail test report with actual vs. expected values.

**Track Bugs & Feature Requests**
Enable error tracking to automatically capture console errors, unhandled exceptions, failed network requests, and CSP violations. Annotate errors as bug reports with severity levels, or create feature requests tied to specific page elements. Export everything as a styled HTML report or JSON.

**Record Video Clips**
Capture short video recordings of your active tab (up to 5 minutes) and download them as WebM files — perfect for attaching to bug reports.

**100% Local & Private**
All data stays in your browser. Sentinel makes zero network requests, has no analytics, and sends nothing to external servers. Your sessions, screenshots, and reports never leave your machine unless you explicitly export them.
